## What is Blindfold?

**Blindfold protects sensitive data (PII) before it reaches AI providers like OpenAI or Anthropic.**

Automatically detect and anonymize 50+ types of personally identifiable information - names, emails, SSNs, credit cards, and more.

## Why Blindfold?

**Compliance**: Meet GDPR, HIPAA, and EU AI Act requirements
**Simple**: One API call to protect all PII
**Reversible**: Restore original data when needed

## How It Works

```
Your App → Blindfold → AI Provider
           (removes PII)
```

1. Send text with sensitive data
2. Get back protected text
3. Send safe text to AI
4. Restore original data (optional)

## Quick Start

<Steps>
  <Step title="Create Account">
    Sign up at [app.blindfold.dev](https://app.blindfold.dev)
  </Step>
  <Step title="Get API Key">
    Generate your API key in the dashboard
  </Step>
  <Step title="Install SDK">
    ```bash
    pip install blindfold-sdk
    ```
  </Step>
  <Step title="Protect Data">
    ```python
    from blindfold import Blindfold
    client = Blindfold(api_key="your-key")
    result = client.tokenize("John Doe, john@example.com")
    # Result: "<PERSON_1>, <EMAIL_1>"
    ```
  </Step>
</Steps>

### Using the API

<Tabs>
  <Tab title="Python">
    ```python
    from blindfold import Blindfold

    # EU region
    client = Blindfold(api_key="your-key")
    ```
  </Tab>

  <Tab title="JavaScript">
    ```javascript
    import { Blindfold } from '@blindfold/sdk';

    // EU region
    const client = new Blindfold({ apiKey: 'your-key' });
    ```
  </Tab>

  <Tab title="cURL">
    ```bash
    # EU region
    curl -X POST https://api.blindfold.dev/api/public/v1/tokenize \
      -H "X-API-Key: your-key" \
      -d '{"text": "Sample text"}'
    ```
  </Tab>
</Tabs>
